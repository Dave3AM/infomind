HTML Basics

HTML (HyperText Markup Language) is the foundation of all websites. It defines the structure and content.

Basic HTML Structure

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>My First Page</title> </head> <body> <h1>Hello World!</h1> <p>This is my first webpage.</p> </body> </html>

Common HTML Tags

Tag Purpose Example
<h1> to <h6> Headings <h1>Main Title</h1>
<p> Paragraph <p>Some text</p>
<a> Link <a href="url">Click</a>
<img> Image <img src="pic.jpg" alt="desc">
<ul> <li> List <ul><li>Item</li></ul>
<div> Container <div>Content</div>

Example: Simple Page

<!DOCTYPE html> <html> <head> <title>About Me</title> </head> <body> <h1>About Me</h1> <h2>Introduction</h2> <p>Hi, I'm a high school student learning programming.</p> <h2>My Interests</h2> <ul> <li>Programming</li> <li>Math</li> <li>Technology</li> </ul> <h2>Contact</h2> <p>Email: <a href="mailto:test@example.com">test@example.com</a></p> </body> </html>
