C++ Quick Reference

Essential syntax and functions at a glance

Basic Syntax

Program Structure

#include <iostream> using namespace std; int main() { // Your code here return 0; }

Comments

// Single line comment /* Multi-line comment */

Output

cout << "Hello"; // Print text cout << variable; // Print variable cout << a << " " << b; // Multiple values cout << endl; // New line

Input

int x; cin >> x; // Read one value cin >> a >> b; // Read multiple values

Data Types

Variables

int age = 17; float price = 9.99; double pi = 3.14159; char grade = 'A'; bool passed = true; string name = "John";

Constants

const int MAX = 100; const float PI = 3.14;

Operators

Arithmetic

+ - * / % // Basic operations ++ // Increment by 1 -- // Decrement by 1 += -= *= // Compound assignment

Comparison

== != // Equal, Not equal < > // Less, Greater <= >= // Less or equal, Greater or equal

Logical

&& // AND || // OR ! // NOT

Control Flow

If Statement

if (condition) { // code } else if (condition) { // code } else { // code }

Switch

switch(variable) { case 1: // code break; case 2: // code break; default: // code }

Loops

For Loop

for(int i = 0; i < n; i++) { // code }

While Loop

while(condition) { // code }

Do-While Loop

do { // code } while(condition);

Loop Control

break; // Exit loop continue; // Skip to next iteration

Arrays

Declaration

int arr[5]; // Empty array int arr[5] = {1,2,3,4,5}; // With values int arr[] = {1,2,3}; // Auto size

Access Elements

arr[0] = 10; // Set value int x = arr[0]; // Get value

Common Operations

// Loop through array for(int i = 0; i < n; i++) { cout << arr[i]; } // Find size int size = sizeof(arr) / sizeof(arr[0]);

Strings

String Operations

string s = "Hello"; s.length() // Get length s[0] // Access character s + " World" // Concatenation s.substr(0,3) // Substring s.find("ll") // Find substring

C-Style Strings

char str[100]; strlen(str) // Length strcpy(s1, s2) // Copy strcat(s1, s2) // Concatenate strcmp(s1, s2) // Compare

Functions

Function Declaration

returnType name(parameters) { // code return value; } // Example int add(int a, int b) { return a + b; }

Void Function

void printMessage() { cout << "Hello"; // No return }

Math Functions

Common Math

#include <cmath> abs(x) // Absolute value pow(x, y) // x to power y sqrt(x) // Square root ceil(x) // Round up floor(x) // Round down max(a, b) // Maximum min(a, b) // Minimum

Common Patterns

Swap Two Variables

int temp = a; a = b; b = temp;

Find Max in Array

int max = arr[0]; for(int i = 1; i < n; i++) { if(arr[i] > max) { max = arr[i]; } }

Sum of Array

int sum = 0; for(int i = 0; i < n; i++) { sum += arr[i]; }

Check Even/Odd

if(num % 2 == 0) { // Even } else { // Odd }

Reverse Array

for(int i = 0; i < n/2; i++) { int temp = arr[i]; arr[i] = arr[n-1-i]; arr[n-1-i] = temp; }

Tips & Tricks

Array Index

Arrays start at index 0. Last element is at n-1.

Integer Division

5/2 = 2 (not 2.5). Use floats for decimals.

Semicolons

Don't forget semicolons after statements!

Case Sensitivity

C++ is case sensitive. 'a' 'A'

Comparison

Use == for comparison, = for assignment

Braces

Always match opening and closing braces
