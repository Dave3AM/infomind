Introduction to C++

C++ is a powerful programming language used for system programming, game development, and competitive programming. It's an extension of the C language with additional features like object-oriented programming.

Your First Program

#include <iostream> using namespace std; int main() { cout << "Hello World!"; return 0; }

Explanation:

  • #include <iostream> - imports input/output library
  • using namespace std; - allows us to use cout without std:: prefix
  • int main() - the main function where program starts
  • cout - prints output to screen
  • return 0; - ends the program successfully

Variables and Data Types

Variables are containers for storing data values. In C++, you must declare the variable type.

Common Data Types

Type Description Example
int Integer numbers int age = 17;
float Decimal numbers float price = 9.99;
double Large decimal numbers double pi = 3.14159;
char Single character char grade = 'A';
bool True or False bool passed = true;
string Text string name = "John";
