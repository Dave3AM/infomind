Sum of Array Elements

Easy

Problem Statement

Given an array of n integers, calculate and print the sum of all elements.

Input Format

  • First line: an integer n (1 n 1000)
  • Second line: n integers separated by spaces

Output Format

Print a single integer - the sum of all array elements.

Examples

Input: 
5
1 2 3 4 5
Output: 
15
Input: 
4
10 -5 3 7
Output: 
15

Solution Explanation

This problem requires basic array traversal and accumulation:

  1. Read the number of elements n
  2. Read n elements into an array
  3. Initialize a sum variable to 0
  4. Loop through array and add each element to sum
  5. Print the final sum

C++ Solution

#include <iostream> using namespace std; int main() { int n; cin >> n; int arr[n]; // Read array elements for(int i = 0; i < n; i++) { cin >> arr[i]; } // Calculate sum int sum = 0; for(int i = 0; i < n; i++) { sum += arr[i]; } // Output result cout << sum; return 0; }

Step by Step

For input: 5, [1, 2, 3, 4, 5]

  • Initialize sum = 0
  • i=0: sum = 0 + 1 = 1
  • i=1: sum = 1 + 2 = 3
  • i=2: sum = 3 + 3 = 6
  • i=3: sum = 6 + 4 = 10
  • i=4: sum = 10 + 5 = 15
  • Output: 15

Time Complexity

O(n) - We traverse the array once

Space Complexity

O(n) - We store n elements in the array

Hints

Hint 1

You need to use a loop to go through each element of the array.

Hint 2

Initialize a variable to 0 before the loop starts, then add each array element to it.

Hint 3

Use the += operator to add each element: sum += arr[i];
